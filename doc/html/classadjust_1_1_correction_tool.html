Detailed Description

Класс предоставляет инструменты для настройки и коррекции свойств 
изображения, включая уровни, кривые и цветовой баланс.

Member Function/Procedure/Process Documentation

◆ adjust_color_balance()

adjust_color_balance ( self,
image,
shadows,
midtones,
highlights ) 
Корректирует цветовой баланс изображения путем настройки теней, 
средних тонов и светлых участков.

Аргументы:
    image (Image): Изображение для настройки.
    shadows (tuple): Кортеж, представляющий настройку цвета для теней.
    midtones (tuple): Кортеж, представляющий настройку цвета для 
                      средних тонов.
    highlights (tuple): Кортеж, представляющий настройку цвета для 
                        светлых участков.

Точно настраивает тональное распределение изображения с

◆ adjust_levels()

adjust_levels ( self,
image,

Изменяет тональный диапазон изображения путем сопоставления 
входных уровней с выходными уровнями.

Аргументы:
    image (Image): Изображение для настройки.
    input_levels (tuple): Кортеж, представляющий диапазон входных уровней.
    output_levels (tuple): Кортеж, представляющий желаемый диапазон 
                          выходных уровней.
